HTML

  • Head elements contain all the metadata for the application.
  • Body element contains all the visible content for the application.
  • The ul tag means unordered list and the li tags are indvidual list iteams.

CSS

  • Htmlcolorcodes.com is a good website for picking out colors by their hexadecimal code.
  • Margin pushes elements as is but padding adds spacing within the element to move it but also make it bigger.

Git

  • Git pull orgin main fetches other commits so you can then merge the branches into the master/main branch.
  • Git status tells you what branch you are currently working in.
  • Git add -A to save all the changes to that branch(commits).
  • Git -m adds a message!

JavaScript

  • Functions need to be called in JavaScript to be used.
  • Console.log() will display whatever is in that parameter into the console.
